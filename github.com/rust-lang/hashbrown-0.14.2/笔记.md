
### 标志
最高位一定是 1
0xFF  empty
0x80  deleted

high7 = (hashcode >> 57) & 0x7F
low57 = hashcode

### 在 group 之间，采用三角数跳跃
```rust
/// <https://fgiesen.wordpress.com/2015/02/22/triangular-numbers-mod-2n/>
struct ProbeSeq { //三角数,   (k*(k+1))/2,  才是是用于在group之间跳跃
    pos: usize,  // 取hashcode的低N位, 本质上就是桶长度取模
    stride: usize,  //每次加16，看起来是跳跃的累计数
}
```
